# Patterns

| Pattern | Use | Output |
|---|---|---|
| Orient | Understand the current bundle before changing it. | Links to relevant MOCs and source notes. |
| Clarify | Convert ambiguous input into a concrete task. | Project note or inbox capture. |
| Execute | Make the change with minimal unrelated churn. | Updated concepts and indexes. |
| Verify | Check structure, links, and conformance. | Verification note or final summary. |
| Handoff | Preserve state for future continuation. | Context packet or handoff note. |

# Agent Defaults

* Read [Vault Charter](/6-System/vault-charter.md) first when the work affects structure.
* Read the relevant MOC before editing a domain.
* Preserve user-authored content unless asked to rewrite it.
* Make decisions visible in [Decisions and Context MOC](/5-MOCs/decisions-and-context-moc.md).
