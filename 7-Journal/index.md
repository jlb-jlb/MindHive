# Journal

Dated daily notes for logs, reflections, planning, and running thoughts. Each note is named by date (`YYYY-MM-DD.md`) and uses `type: Journal`. Journal notes are durable, not inbox captures: use [0-Inbox](/0-Inbox/) for unprocessed items that still need routing.
