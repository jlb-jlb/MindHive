# Definition

PARA organizes information by actionability:

| Category | Meaning | Question |
|---|---|---|
| Projects | Active outcomes with a finish line | What am I trying to complete? |
| Areas | Ongoing responsibilities and standards | What must be maintained? |
| Resources | Reusable reference material | What may be useful later? |
| Archive | Inactive material retained for history | What is no longer active? |

# Local Rules

* Projects need a next action and a done condition.
* Areas need a standard to maintain.
* Resources need reusable value.
* Archive keeps context without competing for attention.
