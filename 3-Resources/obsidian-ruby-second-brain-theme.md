# Summary

MindHive includes a local Obsidian theme named `Ruby Second Brain`. It uses ruby as the accent color while keeping the reading surface calm for long-form notes, MOCs, PARA reviews, and agent handoffs.

# Files

* [Theme CSS](/.obsidian/themes/Ruby%20Second%20Brain/theme.css)
* [Theme Manifest](/.obsidian/themes/Ruby%20Second%20Brain/manifest.json)
* [Appearance Configuration](/.obsidian/appearance.json)

# Design Notes

* Ruby is used for active states, links, tags, headings, checkboxes, and selected navigation.
* The light theme uses warm off-white surfaces.
* The dark theme uses near-black neutral surfaces with ruby highlights.
* Corners are restrained to match a working knowledge system rather than a decorative landing page.
