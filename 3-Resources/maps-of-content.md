# Definition

A Map of Content is a curated navigation note. It explains what exists, why it matters, and how notes relate.

# Local Rules

* A MOC should guide decisions, not merely list files.
* A MOC should link to concepts using bundle-relative links.
* A MOC should be updated when a related concept becomes important.
* A MOC can span PARA boundaries when the thinking naturally spans them.

# MOC Types

| Type | Use |
|---|---|
| Structural MOC | Navigates a section, such as projects or resources. |
| Topic MOC | Navigates a domain, such as agent collaboration. |
| Decision MOC | Collects decisions, assumptions, and context packets. |
