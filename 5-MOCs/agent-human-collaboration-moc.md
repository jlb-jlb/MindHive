# Purpose

This map helps a human and an agent share context without relying on memory hidden inside a chat thread.

# Collaboration Flow

1. Capture unclear input in [Inbox](/0-Inbox/).
2. Promote actionable work into [Projects](/1-Projects/).
3. Promote recurring responsibility into [Areas](/2-Areas/).
4. Store reusable knowledge in [Resources](/3-Resources/).
5. Move dormant material to [Archive](/4-Archive/).
6. Update the relevant MOC and links after every meaningful change.
