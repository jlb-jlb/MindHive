# Project Rules

A project belongs here when it has:

* A desired outcome.
* An owner or responsible actor.
* A next action.
* A clear done condition.

# Project Links

Every project should link to:

* One or more areas that explain why the project matters.
* Resources used to execute the work.
* Decision notes created during the work.
* A handoff note when an agent may resume the work later.
